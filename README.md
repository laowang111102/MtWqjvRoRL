# Python医药销售管理系统 python483

本项目是基于Python的医药销售管理系统，集成了源码、项目文档、PPT及运行教程。以下是对该系统的详细介绍。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

- 商品信息管理
- 销售信息管理
- 用户信息管理
- 供应商信息管理
- 员工信息管理
- 药品销量数据分析
- 药品收入数据分析
- 业绩数据分析

## 系统角色

- 管理员：负责系统整体管理
- 员工：进行日常医药销售操作
- 供应商：管理供货信息

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x

## 部署步骤

1. 准备Python运行环境
2. 安装依赖库
3. 初始化数据库
4. 配置系统参数
5. 运行应用
